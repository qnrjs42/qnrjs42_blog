# map()

forEach()처럼 시맨틱 접근
forEach()처럼 사용

참고 Link: <a href='https://qnrjs42_blog/javascript/Array_ForEach' target='_blank' rel='noreferrer noopener'>qnrjs42_blog/javascript/Array_ForEach</a>

map()은 filter()와 달리<br />
배열의 엘리먼트를 하나씩 읽어가면서<br />
콜백 함수에서 반환한 값을 새로운 배열에 첨부하여 반환한다.

## 1. 반환한 값을 배열에 첨부

```javascript
var value = [10, 20, 30];
var fn = function (el, index, all) {
  return el + this.add;
};

var point = { add: 100 };
var result = value.map(fn, point); // 파라미터: 콜백 함수, this로 참조할 오브젝트
console.log(result);
```

```shell
실행결과
[110, 120, 130]
```

```shell
실행결과 설명

return el + this.add;

리턴할 때
현재 엘리먼트 값과 this 참조 오브젝트를 더해서 반환한다.
```
